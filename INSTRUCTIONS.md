# OLM Cheat V1.0

Script hỗ trợ trả lời câu hỏi trên **olm.vn** .

---

## Features

* **Auto answering** – tự động gửi câu hỏi tới AI và hiển thị câu trả lời
* **Stop timer** – dừng thời gian bài thi
* **Screenshot mode** – hỗ trợ câu hỏi dạng ảnh (`Ctrl + Space`)
* **Manual input** – nhập câu hỏi thủ công nếu script không nhận diện được
* **Custom knowledge** – dùng file `knowledge.txt` để cung cấp câu trả lời sẵn

---

## Installation

1. Tải `Release.zip` từ **Releases**
2. Giải nén và chạy:trainer.exe


3. Nhập **Groq API key** (hoặc Enter để dùng key mặc định)

Script sẽ tạo file:lib/olm_cheat_trained.js

Mở file và **copy toàn bộ nội dung**.

---

## Usage

1. Mở **olm.vn**
2. Nhấn **F12** → tab **Console**
3. Nhập: allow pasting


4. Dán script và nhấn **Enter**

Nếu chạy thành công, UI của script sẽ xuất hiện.

---

## Custom Knowledge

Bạn có thể thêm file:
knowledge.txt


Ví dụ:
Jerry 67 best

Script sẽ ưu tiên dùng dữ liệu trong file này.
